PORTIFÓFIO BACK END

1. Renomeador de imagens com leitura OCR

Problema:

    Os técnicos da empresa precisavam renomear manualmente dezenas de fotos com o horário exato de captura antes de entregá-las aos clientes, o que era demorado e sujeito a erros.

Solução:

    Criei um executável com interface gráfica que permite ao usuário selecionar uma pasta e processa todas as imagens com o Tesseract OCR. A hora é extraída diretamente da imagem e usada para renomear os arquivos automaticamente no formato "hh.mm.ss", com cópias organizadas em uma nova pasta.

Técnologias:

    Python (script principal), PyQt6 (interface gráfica), Tesseract OCR (extração de texto das imagens), PIL/Pillow (leitura de imagem), OS e shutil (operações com arquivos).

2. Compilador de relatórios Word

Problema:

    Ao final de cada solicitação de serviço, era necessário compilar manualmente todos os relatórios escritos e fotográficos de visitas técnicas, alternando os arquivos em ordem cronológica. Esse processo era repetitivo e custava muito tempo.

Solução:

    Automatizei a leitura dos arquivos Word, organizando os relatórios por visita e intercalando os escritos com os fotográficos. Um documento final era gerado automaticamente com todos os relatórios compilados e formatados de forma padronizada para entrega ao cliente.

Técnologias:

    Python (lógica de automação), python-docx (manipulação de documentos Word), OS e pathlib (varredura de arquivos), shutil (cópia e organização dos arquivos).

3. Sistema administrativo com banco local e nuvem

Problema:

    A empresa precisava de um sistema centralizado para controlar dados de funcionários, veículos, viagens e folha de ponto. O processo anterior era dividido entre planilhas e controles manuais, gerando atrasos e erros.

Solução:

    Desenvolvi um sistema desktop com interface gráfica que automatiza o controle administrativo. O sistema possui banco de dados local (SQLite) com sincronização para a nuvem (Supabase), integração com a API do Tangerino para controle de ponto, roteirização com mapas interativos, e geração automática de folhas de ponto em Excel.

Técnologias:

    Python (lógica e automações), PyQt6 (interface gráfica), SQLite3 (banco local), Supabase (banco em nuvem via API), requests (comunicação HTTP), openpyxl e win32com (manipulação de Excel), Folium (mapas), threading (processamento paralelo), psycopg2 (conexão com banco PostgreSQL).

4. Mapeamento de dados sociais para ONG REPLE SOCIAL

Problema:

    A ONG precisava de uma análise territorial detalhada sobre o Complexo da Penha (RJ) para iniciar seu trabalho social com base em dados confiáveis. No entanto, os dados estavam espalhados, incompletos e sem visualização geográfica.

Solução:

    Foi desenvolvido um sistema de coleta, tratamento e visualização de dados públicos. Os dados foram organizados e completados, com preenchimento automático de coordenadas faltantes. Foram criados mapas interativos com pontos georreferenciados, facilitando a análise do território e a tomada de decisão.

Técnologias:

    Python (para manipulação e automação), Pandas e NumPy (para limpeza e organização dos dados), GeoPandas e Shapely (para análise geoespacial), Nominatim (para geocodificação), e Folium (para geração de mapas interativos com HTML e JavaScript embutidos).
